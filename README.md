# registration
The program is developed based on MATLAB 2024a and can be executed directly by running main.m.

Due to data privacy restrictions, the procedures for importing DICOM files, performing vessel segmentation, and related preprocessing steps are not included in this repository. The processed 3D vessel data are provided as VesData3D.m, while the adjacent 2D frames are stored in the Ves2D folder.

After running the program, the rigid registration results will be displayed first. Subsequently, the non-rigid registration results for each frame during the tracking process will be presented. Finally, the overall tracking result will be visualized as an animated sequence.
